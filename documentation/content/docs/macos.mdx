title: macOS

---

## Installation

```bash
curl -L mac.jfa.dev | bash
```

This installs:

- Homebrew-managed casks from `packages/aarch64-darwin/Brewfile`
- zerobrew-managed formulae from `packages/aarch64-darwin/ZeroBrewfile`
- nix-darwin/Home Manager configuration from `packages/aarch64-darwin`

## Package Management

```bash
# Apply Homebrew casks and zerobrew formulae from the repo
or sync

# Sync manifests, then upgrade installed casks and zerobrew packages

```

## Nix-darwin Profile Management

```bash
# Build and apply changes
or switch
or build
or test

# Dry-run changes
or dry

# List and rollback generations
drl
drr
```

Traditional `nix-darwin` commands from the repository directory:

```bash
darwin-rebuild switch --flake ".#macos"
darwin-rebuild build --flake ".#macos"
```
