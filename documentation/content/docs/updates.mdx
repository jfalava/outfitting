---
title: Updates
description: Update workflows after installation for WSL, macOS, and Windows.
---

## WSL

Quick update:

```bash
curl -L wsl.jfa.dev | bash
```

Update local configuration after editing files in `~/.config/outfitting/repo`:

```bash
git pull
hm-sync
```

Update packages:

```bash
hm-update
update-all
```

Re-run post-install work, including the Cloudflare Access-protected licensed
font installation:

```bash
curl -L mac.jfa.dev/post-install | bash
```

Other update modes:

```bash
curl -L wsl.jfa.dev | bash -s -- --update-only
curl -L wsl.jfa.dev | bash -s -- --full-install
```

## macOS

Update local configuration after editing files in `~/.config/outfitting/repo`:

```bash
git pull
o sync
hm-sync
```

Update packages:

```bash
o upgrade
hm-update
update-all
```

## Windows

Update PowerShell profile:

```powershell
irm win.jfa.dev/config/pwsh-profile | iex
```

Install or update Bun global packages:

```powershell
irm win.jfa.dev/bun | iex
```

Re-run post-install work, including the Cloudflare Access-protected licensed
font installation. Run this from an elevated PowerShell session:

```powershell
irm win.jfa.dev/post-install | iex
```

## Install or Update Bun Global Packages (WSL/macOS)

WSL:

```bash
curl -fsSL wsl.jfa.dev/packages/bun | xargs -I {} bun install -g {}
```

macOS:

```bash
curl -fsSL mac.jfa.dev/packages/bun | xargs -I {} bun install -g {}
```
