For Ubuntu-based WSL distributions:

## Installation Modes

Default mode:

```bash
curl -L wsl.jfa.dev | bash
```

- Updates Nix channels and packages
- Applies the Home Manager configuration
- Skips APT package installation

Full installation:

```bash
curl -L wsl.jfa.dev | bash -s -- --full-install
```

Other modes:

```bash
curl -L wsl.jfa.dev | bash -s -- --update-only
curl -L wsl.jfa.dev | bash -s -- --nix-only
```

## Home Manager Commands

```bash
# Apply config changes
hm-sync

# Update packages and apply config
hm-update
```
